## ✨ Features
Quickly supports:
- 🆓 Free.
- 🤫 Quiz cards.
- 📦 Modules.
- 📁 Folders.
- 📝 Tests
  - ✔️ True / False
  - 🔗 Match
  - 🔘 Choose
  - ⌨️ Input
- 🎯 Match mode.
- 🖼️ Image support.
